HTML/ SEO/ meta 태그
● HTML / SEO

<meta>

검색엔진과 소셜미디어에 페이지 정보를 전달하는 <head> 태그

✦언제 쓰는가

🔎
구글 검색결과에 제목/설명 표시 (SEO)

description 메타태그는 검색결과 스니펫에 표시된다. 직접 순위에는 영향 없지만 클릭률(CTR)에 영향을 준다.

📤
카카오톡/슬랙/트위터 링크 공유 시 미리보기

OG(Open Graph) 태그가 없으면 링크를 공유해도 빈 박스가 뜬다. 포트폴리오, 회사 사이트라면 필수.

📱
모바일 뷰포트 설정

viewport 메타태그 없이는 모바일에서 데스크탑 레이아웃이 축소되어 표시된다. 반응형의 전제 조건.

</>기본 문법

html — 실무 필수 세트
<head>
  <!-- 기본 필수 -->
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  <!-- SEO -->
  <title>페이지 제목 | 사이트명</title>
  <meta name="description" content="150자 이내 페이지 설명. 검색 스니펫에 표시됨.">

  <!-- Open Graph (카카오, 슬랙, 페이스북 미리보기) -->
  <meta property="og:type"        content="website">
  <meta property="og:title"       content="페이지 제목">
  <meta property="og:description" content="설명 텍스트">
  <meta property="og:image"       content="https://example.com/og-image.png">
  <meta property="og:url"         content="https://example.com/page">

  <!-- Twitter Card -->
  <meta name="twitter:card"  content="summary_large_image">
  <meta name="twitter:title" content="페이지 제목">
</head>

✕자주 하는 실수

✕ description 없이 배포
<head>
  <title>My Portfolio</title>
  <!-- description 없음 -->
</head>
✓ description + OG 세트
<meta name="description" content="포트폴리오 설명...">
<meta property="og:title" content="My Portfolio">
<meta property="og:image" content="https://...">
⚠️ 포트폴리오를 카카오톡으로 공유할 때 미리보기가 비어있으면 인상이 나쁘다. OG 태그는 배포 전 필수 체크 항목이다.
