flexbox

1차원(행 또는 열) 방향으로 요소를 정렬하는 레이아웃 모델

✦언제 쓰는가

🔝
헤더/네비게이션 — 좌우 분리

로고(왼쪽) + 버튼(오른쪽) 패턴. justify-content: space-between 한 줄로 해결된다. 실무에서 가장 많이 쓰는 패턴.

⏺️
수직 + 수평 완벽 중앙 정렬

display:flex; align-items:center; justify-content:center. 모달, 로딩 스피너, 빈 상태 화면에 사용.

🏷️
아이콘 + 텍스트 인라인 정렬

버튼 내부 아이콘-텍스트 수직 중앙. display:flex; align-items:center; gap:8px. vertical-align 쓰던 시대는 끝났다.

</>기본 문법

css — 자주 쓰는 패턴
/* ① 헤더: 좌우 분리 */
.header {
  display:         flex;
  justify-content: space-between;
  align-items:     center;
}

/* ② 완벽 중앙 (모달, 빈 화면) */
.center-wrap {
  display:         flex;
  align-items:     center;
  justify-content: center;
  min-height:      100vh;
}

/* ③ 아이콘 + 텍스트 */
.btn {
  display:     inline-flex;
  align-items: center;
  gap:         8px;     /* 간격은 gap으로 */
}

/* ④ 카드 목록 (줄바꿈 허용) */
.card-list {
  display:   flex;
  flex-wrap: wrap;
  gap:       16px;
}
속성주요 값설명
justify-contentflex-start / center / space-between / space-around주축(가로) 정렬
align-itemsflex-start / center / stretch / flex-end교차축(세로) 정렬
flex-directionrow / column / row-reverse주축 방향
flex-wrapnowrap / wrap줄바꿈 여부
gap8px / 1rem / 16px 8px자식 요소 간격 (margin 대체)
flex1 / 0 auto / 2자식 요소 비율 확장

▶라이브 데모

flexbox 인터랙티브 — 값을 바꾸며 결과 확인
A
B (키큰)
C

✕자주 하는 실수

✕ gap 대신 margin으로 간격
.list { display: flex; }
.item { margin-right: 16px; }  /* 마지막 아이템도 margin 생김 */
✓ 부모에 gap 사용
.list {
  display: flex;
  gap:     16px;  /* 마지막 아이템 제외 자동 처리 */
}
⚠️ margin으로 간격을 주면 마지막/첫 요소 처리에 :last-child { margin: 0 } 같은 예외가 필요하다. gap이 훨씬 간결하고 명확하다.
